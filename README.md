# 8-bit SAR ADC using Xschem + ngspice

An 8-bit Successive Approximation Register (SAR) ADC designed and verified using the open-source analog IC design flow with **Xschem** and **ngspice**.

## Features
- SAR Logic
- Transmission Gate
- Bottom-Plate Switch
- Capacitive DAC
- Comparator Integration
- Transient Verification
- FFT Analysis

## Tools
- Xschem
- ngspice

## Performance

| Metric | Value |
|---|---:|
| Resolution | 8-bit |
| SNR | 49.6 dB |
| SNDR | 48.7 dB |
| SFDR | 61.8 dB |
| THD | -56.2 dB |
| ENOB | 7.8 bits |


## Getting Started
Open the schematic files in Xschem and run the supplied ngspice simulations.
